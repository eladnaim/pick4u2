# Project Summary
The "Pick4U" project is a community-driven social delivery service application that connects individuals needing package pickups with local collectors. It enhances the delivery process and fosters community engagement through features such as user registration, real-time communication, payment coordination, and a dedicated collector dashboard. "Pick4U" aims to provide a seamless delivery experience while promoting local connections, particularly in Israel. The latest version, "Pick4U v2," includes significant improvements in user experience, bug fixes, and enhanced functionality for both users and collectors.

# Project Module Description
The "Pick4U" app includes the following core features:
- **User Registration**: Simple registration with avatar selection and personal details.
- **Community/Location Selection**: Users can choose their community or location for service.
- **Pickup Request Creation**: Users can create and manage package pickup requests.
- **Chat Functionality**: Enables communication between requesters and collectors.
- **Payment Coordination**: Facilitates payment processes between users.
- **Rating System**: Users can rate their experiences with collectors.
- **Notifications**: Users receive updates about their requests and community activities.
- **Collector Dashboard**: Allows collectors to view and manage available requests.
- **Login Functionality**: Registered users can log in to access their profiles.
- **Push Notifications**: Users can receive push notifications for important updates.

# Technology Stack
- React, TypeScript, and Vite for front-end development.
- Firebase for authentication, Firestore database, and cloud functions.
- Google Maps API for location services.
- Twilio for SMS notifications.
- Stripe for payment processing.
- OneSignal for push notifications.
- Tailwind CSS for styling.

# Usage
To install dependencies and set up the project, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/[USERNAME]/pick4u-app.git
   cd pick4u-app
   ```
2. Install required dependencies using the package manager:
   ```bash
   pnpm install
   ```
3. Copy the environment variables template:
   ```bash
   cp .env.example .env
   ```
   Update the `.env` file with your Firebase API keys, Google Maps API key, Twilio credentials, etc.
4. Build the application:
   ```bash
   pnpm run build
   ```
5. Run the application:
   ```bash
   pnpm run dev
   ```
